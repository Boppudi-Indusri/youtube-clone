## Tech Stack
Frontend: React (Vite), Axios, React Router  
Backend: Node.js, Express.js  
Database: MongoDB  
Authentication: JWT  

## Features
- User Authentication (JWT)
- Video Listing & Search
- Category Filters
- Video Player Page
- Like & Dislike System
- Comment CRUD
- Channel CRUD
- Responsive Design

## Installation

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm run dev  

## Environment Variables
MONGO_URI=your_mongodb_url  
JWT_SECRET=your_secret
